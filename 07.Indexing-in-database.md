## Without Indexing ##
Without indexing, one will have to scan line by line for every query

## An Index Table : ## 
Acts like a map to store memory address

## Types of indexing methods ##

1. Primary: Main value for uniqueness and fast access
2. Secondary: Use another column with the primary key
3. Composite: Indexing based on multiple columns to identify results
4. Hashing : Using a function to store the data in a calculated bucket #

## Challenges ##
The number of write operations will increase (Update index table too)
Adding indexes is not free, we need more space --> more cost incurred
Cleanup unused indexes periodically(Audit regularly)
